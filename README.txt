미치카츠TV V5.2 업로드 방법

이번 버전은 사용자가 제공한 참고 이미지의 분위기를 바탕으로 다시 구성했습니다.

주요 변경
- 중앙 집중형 시네마틱 오프닝
- 거대한 미치카츠TV 타이틀
- 중앙 감마 타이탄 실루엣
- 좌우 번개와 폭풍 배경
- 오른쪽 유튜브/인스타그램 세로 퀵메뉴
- 오른쪽 아래 음악 상태 위젯
- 하단 브랜드/소셜 카드/실시간 시계 구조
- 기존 음악, 40% 볼륨, 무한 반복, 플레이어, 테마, 갤러리, 영상, 게임, 연락처 기능 유지
- 외부 이미지와 assets 폴더 미사용

업로드
1. ZIP 압축을 풉니다.
2. GitHub hulk 저장소에서 Add file → Upload files
3. index.html, style.css, script.js, music 폴더, README.txt 업로드
4. Commit changes
5. 1~2분 후 Ctrl+F5
