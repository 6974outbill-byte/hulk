HULK 개인 홈페이지 업로드 방법

1. ZIP 파일의 압축을 풉니다.
2. GitHub의 hulk 저장소로 이동합니다.
3. Add file → Upload files를 누릅니다.
4. index.html, style.css, script.js 파일을 모두 업로드합니다.
5. Commit changes를 누릅니다.
6. 약 30초~2분 후 Cloudflare Pages가 자동 배포합니다.
7. https://hulk-w1v.pages.dev 를 새로고침합니다.

수정 포인트
- 이메일: index.html에서 your@email.com 검색 후 변경
- 소개 문구: ABOUT ME 구역 수정
- 프로젝트: PROJECTS 구역 수정
- 갤러리: GALLERY 구역 수정
