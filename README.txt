미치카츠 V2 업로드 방법

1. ZIP 압축을 풉니다.
2. GitHub hulk 저장소에서 Add file → Upload files를 누릅니다.
3. index.html, style.css, script.js, music 폴더를 모두 업로드합니다.
4. Commit changes를 누릅니다.
5. 약 30초~2분 뒤 hulk-w1v.pages.dev를 새로고침합니다.

음악
- ENTER 클릭 시 볼륨 40%로 시작
- 무한 반복
- 오른쪽 아래 MUSIC 버튼으로 켜기/끄기
