미치카츠TV V5 업로드 방법

이 버전은 외부 배경 이미지와 assets 폴더를 사용하지 않습니다.
따라서 기존처럼 배경 그림과 실제 사이트 화면이 겹치는 문제가 없습니다.

적용 기능
- 미치카츠TV 브랜드 전체 적용
- 영화풍 다크 감마 배경
- 초록 번개와 감마 파티클
- 영화 오프닝 스타일 인트로
- 들어가기 클릭 시 화면 흔들림, 번개, 충격파
- 음악 40% 시작
- 음악 무한 반복
- 음악 ON/OFF 버튼
- 고급 음악 플레이어 UI
- 볼륨 조절 슬라이더
- 유튜브·인스타그램 홍보 카드
- 실시간 시계
- 테마 전환 버튼
- 갤러리·영상·게임·연락처 섹션
- PC·태블릿·모바일 반응형

업로드 순서
1. ZIP 압축을 풉니다.
2. GitHub hulk 저장소에서 Add file → Upload files
3. 아래 항목을 전부 업로드합니다.
   index.html
   style.css
   script.js
   music 폴더
   README.txt
4. Commit changes
5. 30초~2분 뒤 사이트에서 Ctrl+F5

링크 변경
script.js 맨 위 SETTINGS에서 수정합니다.

youtube: 실제 유튜브 주소
instagram: 실제 인스타그램 주소
email: 실제 이메일 주소

기존 assets 폴더는 남아 있어도 V5에서 사용하지 않습니다.
