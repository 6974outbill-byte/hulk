미치카츠TV V4 CLEAN 업로드 방법

이 버전은 외부 배경 이미지나 assets 폴더를 전혀 사용하지 않습니다.
따라서 이전처럼 그림과 실제 화면이 겹치는 문제가 없습니다.

업로드 순서
1. ZIP 압축을 풉니다.
2. GitHub hulk 저장소에서 Add file → Upload files를 누릅니다.
3. 아래 파일과 폴더를 전부 업로드합니다.
   - index.html
   - style.css
   - script.js
   - music 폴더
4. Commit changes를 누릅니다.
5. 약 30초~2분 뒤 사이트에서 Ctrl+F5를 누릅니다.

기존 assets 폴더
- 남아 있어도 V4에서는 사용하지 않습니다.
- 완전히 정리하고 싶다면 assets 폴더 안 파일을 삭제해도 됩니다.
