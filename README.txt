미치카츠TV V5.3 업로드 방법

이번 버전은 사용자가 제공한 참고 이미지를 직접 활용해
첫 화면과 메인 히어로를 참고 이미지 느낌 그대로 보이도록 구성했습니다.

핵심 방식
- 참고 이미지 상단을 hero-reference.jpg로 사용
- 이미지 위에 텍스트를 다시 중복해서 올리지 않음
- 들어가기 / 유튜브 / 인스타그램 영역만 투명 클릭 영역으로 연결
- 스크롤 이후 실제 HTML 콘텐츠 섹션 제공
- 음악, 40% 볼륨, 무한 반복, 플레이어, 시계, 테마, 갤러리, 영상, 게임, 연락처 유지

업로드
1. ZIP 압축을 풉니다.
2. GitHub hulk 저장소에서 Add file → Upload files
3. index.html, style.css, script.js, assets 폴더, music 폴더, README.txt 업로드
4. Commit changes
5. 1~2분 뒤 Ctrl+F5
