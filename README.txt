미치카츠TV V3 업로드 방법

1. ZIP 압축을 풉니다.
2. GitHub의 hulk 저장소로 이동합니다.
3. Add file → Upload files를 선택합니다.
4. 아래 파일과 폴더를 전부 업로드합니다.
   - index.html
   - style.css
   - script.js
   - assets 폴더
   - music 폴더
5. Commit changes를 누릅니다.
6. 약 30초~2분 뒤 Cloudflare Pages 사이트를 새로고침합니다.

V3 주요 기능
- 영화풍 다크 감마 배경
- 첫 입장 화면과 화면 흔들림/번개 효과
- ENTER 클릭 시 음악 40% 재생
- 무한 자동 반복
- 음악 ON/OFF 버튼
- 감마 파티클 효과
- 유튜브/인스타그램 홍보 카드
- 모바일 최적화
- 테마 전환 버튼
- 실시간 시계
- 맨 위로 버튼
- 갤러리, 영상, 게임, 연락처 섹션
